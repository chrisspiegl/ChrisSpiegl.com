Aspect Ratio Calculator

Multiple times per month I ask myself: what aspect ratio is this pixel amount? This is the tool that helps me know better.

Use the form below to calculate the missing value for a particular aspect ratio. This is useful, for example, when resizing photos or video.

Explanation / Instructions

  1. Enter your values into Old Width and Old Heightrespectively.
  2. Enter one of the values of New Width or New Height.
  3. The missing value will be calculated automatically.
  4. You can change all values at any time, the calculation updated automatically.

Formula

You have a photo or video that is 2400 x 3000 pixels in resolution, but you'd like to send a file that is only 400px in width to your client for review. To calculate the new height of your photo you can use the following formula (or the calculator above):

(org. height / org. width) x new width = new height

With the example numbers above, this would be the calculation:

(2400 / 3000) x 400 = 320
